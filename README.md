# IPPIN website replica

This is a first-pass static replica of the current IPPIN Japanese Dining homepage.

Open `index.html` in a browser to view it. The page intentionally uses remote Wix image URLs for the first prototype, so the design can be compared with the live site before replacing assets with owned local files.

The current replica includes local page shells for:

- `whatson.html`
- `menu.html`
- `bottomless.html`
- `beverage.html`
- `functions.html`
- `about.html`
- `gift-card.html`

## Next improvements

- Replace remote image URLs with original image files supplied by IPPIN.
- Replace the JavaScript-injected shared footer with server-side/static templates if the project moves to a build system.
- Expand the full menu page with all a la carte, lunch, and dietary option details.
- Tune spacing and image crops against desktop and mobile screenshots from the live site.
- Connect real SevenRooms, Google Maps, Instagram, and gift card URLs.
